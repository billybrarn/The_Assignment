15 HTML Tags and Their Uses


HTML tags and how to use them are like the basics of web pages.

Basic Structure Tags

  1. < !DOCTYPE>: Defines the document type and version of HTML.

  2. < html>: The root element of an HTML document. < /html > is used to close the element html docmument.

  3. < head>: Contains meta-information about the HTML document.

  4. < title>: Sets the title of the HTML document, displayed in the browser's title bar.

  5. < body>: Contains the content of the HTML document.

    6. Text Formatting Tags

  6. < h1> to < h6>: Define HTML headings, with < h1> being the highest level and < h6> the lowest.

  7. < p >: Defines a paragraph.

  8. < br>: Inserts a line break.

  9. < hr>: Creates a horizontal line (rule) for thematic breaks.

  10. < u>: Defines underlined text.

List Tags

  • < ul >: Defines an unordered (bulleted) list.

  • < ol >: Defines an ordered (numbered) list.

  • < li >: Defines a list item.

  • < a>: Defines a hyperlink.

  • < img>: Embeds an image.
